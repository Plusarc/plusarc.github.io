Rotate Image

You are given an n x n 2D matrix representing an image.

Rotate the image by 90 degrees (clockwise).

Follow up:
Could you do this in-place?

Method

首先要观察出的一点是，一个点(x, y)在顺时针90度旋转后移到的坐标为(y, matrix.length-x).

这样最简单的方法就是建立一个新矩阵，对应原来的矩阵，按照对应位置一一复制，然后再复制回原矩阵。

题中follow up要求do it in-place.

将matrix看成一圈一圈的正方形，最外一圈，内一圈，内一圈……
顺时针90度旋转就是将这些正方形的每条边移动到其顺指针的下一条边，对应的每个点由(x, y)到(y, matrix.length-x).

所以旋转整个矩阵，可以：
依次旋转最外围正方形，再选择内一圈正方形……直到只有边长为1或0。

对于旋转正方形，将这些正方形的每条边移动到其顺指针的下一条边，对应的每个点由(x, y)到(y, matrix.length-x).
