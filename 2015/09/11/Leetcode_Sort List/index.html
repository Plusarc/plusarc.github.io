Sort List

Sort a linked list in O(n log n) time using constant space complexity.

Method

这里使用归并排序，无需开辟新的空间，时间复杂度O(log(n)).

与数组的merge sort不同的是，无法直接通过下标二分数组。

这里使用快慢指针遍历链表获得链表的前半段和后半段，然后分别对前后半段递归排序，再进行Merge。
