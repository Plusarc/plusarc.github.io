Restore IP Addresses

Given a string containing only digits, restore it by returning all possible valid IP address combinations.

For example:
Given "25525511135",

return ["255.255.11.135", "255.255.111.35"]. (Order does not matter)

Method

回溯法求解，对于当前的每个数字，选择和累积的数字创建新的字段，或者不创建新的字段，并入到下一个数字的递归中。

IP地址的条件：这种表示法下，IP地址为32位，4个字段，每个字段范围为0-255。

根据条件进行回溯:

  1. 累计的数字>255，
  2. 当前位置没有到最后一位，但是字段数达到4
  3. 当前为最后一位，但总字段数小余4.
  4. 上一个数字为0的情况。

最后一个回溯条件是为了注意处理IP地址中连续为0的情况。也就是00010，只能解析为0.0.0.10,不存在字段表示为0.0.01.0的情况。
