Single Number II

Given an array of integers, every element appears three times except for one. Find that single one.

Note:
Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Method

Method 1

使用HashMap，一遍扫描数组，如果不在set中，则添加到set中。如果元素已经在set中，且次数为1，则更新为2，如果次数为2，则从set中移除。
按照题目的条件，扫描完后只会剩下一个只出现一遍的元素，取出返回结果

Method 2

使用位操作，尝试消除掉出现3次的数字(参考)。

若一个数字出现3次，则该数字的二进制表示中每个位置的数值为1的出现次数为3次，如：5的二进制表示为101，若5出现3次，则其二进制表示中的第一个位置和第二个位置出现1的次数和均为3。那么对该位1出现的次数对3取余，最终的结果所对应的就是所要求的Single Number。
