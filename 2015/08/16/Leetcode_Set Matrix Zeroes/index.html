Set Matrix Zeroes

Given a m x n matrix, if an element is 0, set its entire row and column to 0. Do it in place.

Method

扫描matrix，将为0的元素的x，y坐标存入rowSet，colSet中，避免部分重复置0操作。如果colSet或rowSet满了，代表整个矩阵需要置0.清空另一个set，跳出扫描操作。

根据rowSet，colSet存储的行，列置0.

更进一步的优化是，设置rowNotZero作为行元素是否全为0的boolean，和colNotZero的boolean数组用于指示每一列元素是否全为0. 如果为false，代表这些行，列的元素已经全为0，无需重置，根据这些指示值相应的移除set中相应的行，列。这里没有给出实现，只需要加一些判断操作。
