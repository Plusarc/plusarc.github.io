Binary Search Tree Iterator

Implement an iterator over a binary search tree (BST). Your iterator will be initialized with the root node of a BST.

Calling next() will return the next smallest number in the BST.

Note: next() and hasNext() should run in average O(1) time and uses O(h) memory, where h is the height of the tree.

Method

next操作相当于取出BST的最小节点并将其从树中删除。一直向左子结点遍历知道其左子结点为null时，即为这个树的左子最小节点，获得该节点值。另外需要将这个节点从BST中删除，还需要保持BST的性质。

若该节点没有子节点，则直接删除。
若该节点有右子节点，其父节点的左指针指向该节点的右子节点。
如果该节点为根节点，需要将根节点重新设置为根的右子节点。

遍历完了树就没了，破坏性太大。。然后时间复杂度是O(h)的，无法达到O(1)。上面的方法一般是用于BST删除最小节点时用的。
