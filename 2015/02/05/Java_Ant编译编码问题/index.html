编码问题来源于代码的注释，所以应该是中文编码的问题，如果是使用eclipse，可以随便打开一个有中文注释的java文件查看properties，确认文件的编码，这次的情况是部分文件为GBK编码。

找到项目的Ant的build.xml，找到javac的配置部分，加入的encoding的指定，否则默认是使用UTF-8进行编译。
