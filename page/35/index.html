Maximum Subarray

Find the contiguous subarray within an array (containing at least one number) which has the largest sum.

For example, given the array [−2,1,−3,4,−1,2,1,−5,4],
the contiguous subarray [4,−1,2,1] has the largest sum = 6.

Method

进阶：矩阵A中有N*N个数，求A的最大的子矩阵。
进阶：枚举最大子矩阵的上下边界x和y，将第x行到第y行每一列的数叠加成为一个数。然后就成为了一个初阶的问题。时间复杂度O(n^3)。

Implement Queue using Stacks

mplement the following operations of a queue using stacks.

push(x) – Push element x to the back of queue.
pop() – Removes the element from in front of queue.
peek() – Get the front element.
empty() – Return whether the queue is empty.

Notes:
You must use only standard operations of a stack – which means only push to top, peek/pop from top, size, and is empty operations are valid.
Depending on your language, stack may not be supported natively. You may simulate a stack by using a list or deque (double-ended queue), as long as you use only standard operations of a stack.
You may assume that all operations are valid (for example, no pop or peek operations will be called on an empty queue).

Method

使用两个栈，一个inStack，一个outStack。push的元素都放入inStack，需要pop或peek时，判断outStack是否为空，若为空则pop所有inStack中的元素并push到outStack中，然后pop或peek outStack的栈顶元素，若不为空，则直接pop或peek outStack的栈顶元素。

第二种方法是每次push时，使用一个辅助数组或辅助栈保存当前栈pop出的所有元素，当前栈为空，然后放入push的元素，再将辅助数组或辅助栈保存的元素按照正确顺序放回当前栈中。

Convert Sorted List to Binary Search Tree

Given a singly linked list where elements are sorted in ascending order, convert it to a height balanced BST.

Method

递归解决，每次使用快慢指针扫描数组，快指针到链表末尾时，慢指针正好处于链表中间。另外慢指针扫描时，同时将扫描过的链表进行翻转。由于慢指针正好处于链表中间，将其取作根节点，同时用同样的方法递归扫描左子链表构建左子树，扫描右子链表构建右子树。时间复杂度O(nlogn)

Word Break

Given a string s and a dictionary of words dict, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

For example, given
s = “leetcode”,
dict = [“leet”, “code”].

Return true because “leetcode” can be segmented as “leet code”.

Method

Naive method

对于target word进行所有可能的拆分，并检测所有拆分部分是否都在dict中。

DP method

问题可以理解为target word中，targetword.substring(0, length)字符串能否由字典中的词构成。子问题为targetword.substring(0, n)字符串能够由字典中的词构成。

如果targetword.substring(0, n)能够由字典中的词构成，则其必定是由可以由字典中的词构成的子串targetword.substring(0, m)(m<n)，加上字典中的一个词构成的。

加上b(n)表示0-n的子串能够由字典组成，状态转移方程可以写为：

b(n) = (b(m) && dict.contains(targetword.substring(m,n))) || dict.contains(targetword)
