Two Sum

Given an array of integers, find two numbers such that they add up to a specific target number.

The function twoSum should return indices of the two numbers such that they add up to the target, where index1 must be less than index2. Please note that your returned answers (both index1 and index2) are not zero-based.

You may assume that each input would have exactly one solution.

Input: numbers={2, 7, 11, 15}, target=9
Output: index1=1, index2=2

Method

Naive method

暴力穷举法，选择第一个元素，依次选择与后面每个元素求和，看是否等于target，再选择第二个元素，依次选择与后面每个元素求和，直到找到solution。复杂度为O(n^2)

loop1: a[0]+a[1]==target? a[0]+a[2]==target? … a[0]+a[n-1]==target?
loop2: a[1]+a[2]==target? a[1]+a[3]==target? … a[1]+a[n-1]==target?
loop..

Two pointer

先对数组进行排序，然后设立左指针left指向第一个元素，右指针right指向最后一个元素。对左右指针元素求和：

  • if sum==target，return solution
  • if sum<target，left++
  • if sum>target，right–

直到right<left, 则没有solution。复杂度O(n*log(n))。

HashMap method

把遍历过的元素及其下标都存入hashmap，对于当前遍历的元素ele，判断(target-ele)是否在hashmap，若存在，则返回solution，若不存在，则将(key=ele, value=pos)存入hashmap。

使用空间换时间的策略，使用hashmap记录所有遍历过的元素，一次遍历。复杂度O(n)。
