Palindrome Number

Determine whether an integer is a palindrome. Do this without extra space.

Method

用 / 操作得到最高位，%操作得到最低位，并比较。然后同样用 % 和 / 处理数字去掉最高最低位。

另一种解法是先转为Char Array，再判断，不过需要额外空间。

Remove Nth Node From End of List

Given a linked list, remove the nth node from the end of list and return its head.

For example,

Given linked list: 1->2->3->4->5, and n = 2.

After removing the second node from the end, the linked list becomes 1->2->3->5.

Note:
Given n will always be valid.
Try to do this in one pass.

Method

设置快慢指针(这里不是指速度不同，而是指开始移动的时间不同)，快指针移动了n步之后，慢指针才开始移动，当快指针移动到链表尾端时，慢指针位于目标节点前，删除该节点。

Merge Two Sorted Lists

Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

Method

设置两个指针指向两个list当前位置，以及一个指针记录新的的list当前的位置。每次选择两个list中当前元素中较小的一个，将新list指针的next指向该元素。设置前同时注意维护下一个指针信息和之后移动指针到下一个。

Add Binary

Given two binary strings, return their sum (also a binary string).

For example,
a = "11"
b = "1"
Return "100".

Method

设置长度为a,b中较长长度的数组digits，按对应位累加两个字符串的位存储存入digits。然后统计digits，位大于1则进位。注意处理最后是否有进位，有则在最后结果字符串先添加1，然后将digits处理后的为添加到结果字符串里。
