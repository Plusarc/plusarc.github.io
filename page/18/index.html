For example, given n = 12, return 3 because 12 = 4 + 4 + 4; given n = 13, return 2 because 13 = 4 + 9.

Word Break II

Given a string s and a dictionary of words dict, add spaces in s to construct a sentence where each word is a valid dictionary word.

Return all such possible sentences.

For example, given
s = "catsanddog",
dict = ["cat", "cats", "and", "sand", "dog"].

A solution is ["cats and dog", "cat sand dog"].

Method

比较直接的解法是直接采用回溯法，但会导致过多的不必要recursion。

动态规划法，不同与Word Break问题，这里需要记录所有可能的solution，而不是是否存在solution。所以代替boolean[]表示是否能到达某个位置，采用List<List<Integer>>记录到达每个位置的可能的前面的所有位置。

最后从后向前递归输出所有能抵达最后位置的路径。

注意下标的处理，Java中的substring截取的参数与位置的不同。
