Word Break II

Given a string s and a dictionary of words dict, add spaces in s to construct a sentence where each word is a valid dictionary word.

Return all such possible sentences.

For example, given
s = "catsanddog",
dict = ["cat", "cats", "and", "sand", "dog"].

A solution is ["cats and dog", "cat sand dog"].

Method

比较直接的解法是直接采用回溯法，但会导致过多的不必要recursion。

动态规划法，不同与Word Break问题，这里需要记录所有可能的solution，而不是是否存在solution。所以代替boolean[]表示是否能到达某个位置，采用List<List<Integer>>记录到达每个位置的可能的前面的所有位置。

最后从后向前递归输出所有能抵达最后位置的路径。

注意下标的处理，Java中的substring截取的参数与位置的不同。

Pow(x, n)

Implement pow(x, n).

Method

最直接的方法是直接将x相乘n次，时间复杂度为O(n).

更高效的是二分法处理，x^n = x^(n/2) * x^(n/2) * x^(n%2)，时间复杂度为O(log(n)).

注意处理，负指数情况以及负指数是Integer.MIN_VALUE的情况。

Valid Sudoku

Determine if a Sudoku is valid, according to: Sudoku Puzzles - The Rules.

The Sudoku board could be partially filled, where empty cells are filled with the character ‘.’.

A partially filled sudoku which is valid.

Note:
A valid Sudoku board (partially filled) is not necessarily solvable. Only the filled cells need to be validated.

Method

本质就是要求每一行不能有重复元素，每一列不能有重复元素，每一个九宫格内不能有重复元素。

关键是如何记录已经使用过的元素和减少遍历次数。

这里使用了两种记录方式，都只需要扫描一遍。

  1. 使用HashSet记录已经使用过的元素。
  2. 由于元素是有限的，都在1-9之间，可以采用boolean数组记录是否已经使用过。

按照行遍历，需要维持 1个行boolean数组(or Set), 3个九宫格boolean数组，9个列boolean数组。
