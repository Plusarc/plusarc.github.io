Rectangle Area

Find the total area covered by two rectilinear rectangles in a 2D plane.

Each rectangle is defined by its bottom left corner and top right corner as shown in the figure.

Method

求两个矩形在平面上占有的面积，关键是抽象出判断两个矩阵是否有重合，如何求出重合的面积，需要仔细观察。

想象随意给两个矩形， 它们有重合的条件是：

  1. 两个矩形下边的较大值必须 小于 两个矩形上边的较小值
  2. 两个矩形左边的较大值必须 小于 两个矩形右边的较小值

Implement Stack using Queues

Implement the following operations of a stack using queues.

push(x) – Push element x onto stack.
pop() – Removes the element on top of the stack.
top() – Get the top element.
empty() – Return whether the stack is empty.

Method

由于queue的性质是先进先出，所有要实现stack的pop操作，必须将queue的前n-1个元素pop出，然后才能得到stack需要pop出的元素。对于queue的pop出的前n个元素，直接将重新push到queue中，这样结构上仍然保持了原来push顺序，且不用开辟新空间。

对于top操作，将整个queue的n个元素pop并push回去，并记录最后一个元素值，作为栈顶元素。

Basic Calculator

Implement a basic calculator to evaluate a simple expression string.

The expression string may contain open ( and closing parentheses ), the plus + or minus sign -, non-negative integers and empty spaces .

You may assume that the given expression is always valid.

Some examples:

Method

一个解决 运算式 计算的通用方法是，先将中缀表达式转为后缀表达式(逆波兰式)，然后再求值，下次详细总结一下。

参考:

支持四则混合运算的计算器
逆波兰表达式的栈实现方式
中缀表达式转后缀表达式
逆波兰式Wiki

步骤：

  1. 先求表达式的后缀表达式
    具体过程为：
    1) 如果遇到操作数，则直接输出。
    2) 如果遇到操作符，则检查栈顶元素优先级，如果其优先级不低于当前操作符（左括号除外），则弹出栈顶元素并输出。重复此过程直到栈顶元素优先级小于当前操作符或为左括号，或者栈为空。 然后将当前操作符压入栈中。
    3）如果遇到左括号，直接压入栈中。
    4) 如果遇到右括号，则将栈中元素弹出，直到遇到左括号为止。左括号只弹出栈而不输出。
    5) 表达式处理完毕，则将栈中元素依次弹出。

注意只有遇到右括号的情况下才会弹出左括号，其他情况都不会弹出。

例如，上述中缀表达式 a + b c - (d - e) f 的转换过程为：

操作 res 栈
1) 遇到操作数a, 直接输出a a
2) 遇到操作符+, +压栈 a +
3) 遇到操作数b, 直接输出b ab +
4) 遇到操作符*, 压栈 ab + *
5) 遇到操作数c, 直接输出 abc + *
6) 遇到操作符-, 栈顶弹出并输出，然后-进栈 abc* + -
7）遇到操作符(,直接进栈 abc* + - (
8) 遇到操作数d,直接输出 abc*d + - (
9) 遇到操作符-，压栈 abc*d + - ( -
10) 遇到操作数e，直接输出 abc*de + - ( -
11）遇到操作符),(之后全部输出 abc*de- + -
12）遇到操作符*,压栈 abc*de- + - *
13) 遇到操作数f,输出 abc*de-f + - *
14) 处理完毕，全部出栈输出。 abcde-f-+

2.基于后缀表达式求值。

中缀表达式5 + ((1 + 2) * 4) − 3写作5 1 2 + 4 * + 3 −.
下表给出了该逆波兰表达式从左至右求值的过程，堆栈栏给出了中间值，用于跟踪算法。

输入 操作 堆栈 注释
5 入栈 5
1 入栈 5, 1
2 入栈 5, 1, 2
+ 加法运算 5, 3 (1, 2)出栈；将结果（3）入栈
4 入栈 5, 3, 4
* 乘法运算 5, 12 (3, 4)出栈；将结果（12）入栈
+ 加法运算 17 (5, 12)出栈；将结果 (17)入栈
