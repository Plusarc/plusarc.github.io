Clone Graph

Clone an undirected graph. Each node in the graph contains a label and a list of its neighbors.

OJ’s undirected graph serialization:
Nodes are labeled uniquely.

We use # as a separator for each node, and , as a separator for node label and each neighbor of the node.
As an example, consider the serialized graph {0,1,2#1,2#2,2}.

The graph has a total of three nodes, and therefore contains three parts as separated by #.

First node is labeled as 0. Connect node 0 to both nodes 1 and 2.
Second node is labeled as 1. Connect node 1 to node 2.
Third node is labeled as 2. Connect node 2 to node 2 (itself), thus forming a self-cycle.
Visually, the graph looks like the following:

   1
  / \
 /   \
0 --- 2
     / \
     \_/

Method

DFS遍历并复制，建立HashMap映射记录已经复制过的节点，以避免重复复制。

Triangle

Given a triangle, find the minimum path sum from top to bottom. Each step you may move to adjacent numbers on the row below.

For example, given the following triangle

The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

Note:
Bonus point if you are able to do this using only O(n) extra space, where n is the total number of rows in the triangle.

Method

动态规划，子问题是以当前层各个节点结尾的最短路径。以最后一层的各个节点结尾的路径的最短路径值中，最小值即是问题的解。

triangle[i][j]处理后保存的是到达第i层第j个节点的最短路径值。(也就是原来list<list<Integer>>，这里使用数组比较直观。)

到达每个点的路径的上一层节点只有两个选择，就是上一层与当前节点相邻的两个节点。(开头与末尾节点只有一种可能)，从中选择较短路径。

Restore IP Addresses

Given a string containing only digits, restore it by returning all possible valid IP address combinations.

For example:
Given "25525511135",

return ["255.255.11.135", "255.255.111.35"]. (Order does not matter)

Method

回溯法求解，对于当前的每个数字，选择和累积的数字创建新的字段，或者不创建新的字段，并入到下一个数字的递归中。

IP地址的条件：这种表示法下，IP地址为32位，4个字段，每个字段范围为0-255。

根据条件进行回溯:

  1. 累计的数字>255，
  2. 当前位置没有到最后一位，但是字段数达到4
  3. 当前为最后一位，但总字段数小余4.
  4. 上一个数字为0的情况。

最后一个回溯条件是为了注意处理IP地址中连续为0的情况。也就是00010，只能解析为0.0.0.10,不存在字段表示为0.0.01.0的情况。

Populating Next Right Pointers in Each Node

Given a binary tree

struct TreeLinkNode {
  TreeLinkNode *left;
  TreeLinkNode *right;
  TreeLinkNode *next;
}

Populate each next pointer to point to its next right node. If there is no next right node, the next pointer should be set to NULL.

Initially, all next pointers are set to NULL.

Note:
You may only use constant extra space.
You may assume that it is a perfect binary tree (ie, all leaves are at the same level, and every parent has two children).

Method

Method 1

由于条件是一棵perfect，每个节点都有确定的序号，并且可以确定是否是否右驱节点，如果有，其右驱节点序号为当前节点序号+1。

这样最直接的方法是建立一个数组，遍历树一遍，将节点的引用存入数组对应的位置(序号位置)。然后再遍历树一遍，设置next的值。

需要O(n)的空间。

后面又想了下面的方法2，无需额外空间，实现也更简洁，所以就没有实现Method 1，这里只给出了Method 2的实现.

Method 2

对于一个节点如果左右子节点非空，则设置root.left.next = root.right.

另外如果其next!=null，则设置root.right.next = root.next.left，也就是其右子节点的next指向其next节点的左子结点。

递归地对左右子树进行同样操作。
