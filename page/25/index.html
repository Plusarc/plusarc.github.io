Permutations II

Given a collection of numbers that might contain duplicates, return all possible unique permutations.

For example,
[1,1,2] have the following unique permutations:
[1,1,2], [1,2,1], and [2,1,1].

Method

DFS递归搜索，把问题看做是在第1，2，3…放置数的问题，首先选择第一个位置的数，递归选择第二个位置的数…在Permutations I中，只需记录已经使用过了的数字在原数组中的位置，这样在下一步选择数字时不能再使用这些数字。这个问题中，由于数组允许重复值的存在，因此在选择当前位置的数字时，也需要记录这个位置已经使用过的，如果已使用过，则无需再搜索一遍，因为其结果是相同的。

Jump Game

Given an array of non-negative integers, you are initially positioned at the first index of the array.

Each element in the array represents your maximum jump length at that position.

Determine if you are able to reach the last index.

For example:
A = [2,3,1,1,4], return true.

A = [3,2,1,0,4], return false.

Method

Method 1

会超时，参考accept方案的请直接参考method2，这里只是记录自己的思考过程。

动态规划，设置一个boolean数组reach，长度与nums长度相同，每个位置值记录对应位置能否到达终点。

核心是，若一个位置的可达到范围内 有 能抵达终点的点，则这个位置也能抵达终点。从终点向前考虑，依次确定每个位置是否能到达终点。最后返回 reach[0]

Method 2

同样是动态规划，本质上跟Method 1是相同的，直接记录当前能到达的最远位置。

Decode Ways

A message containing letters from A-Z is being encoded to numbers using the following mapping:

Given an encoded message containing digits, determine the total number of ways to decode it.

For example,
Given encoded message "12", it could be decoded as "AB" (1 2) or "L" (12).

The number of ways decoding "12" is 2.

Method

DFS solution

递归解法，深度有限搜索可行解的个数，每次可能往后面跳一位，或者跳两位。
有比较多的重复计算，会超时。

DP solution

动态规划。

如果一个数字不能和后一个数字组合，则从后面到这个数字的组合数跟到前一个数字相同，也就是res[i]=res[i+1]。
如果一个数字可以和后一个数字组合，则其数量由两种情况组成，一种是这个数字与后一个数字组合，一种是这个数字与后面一个数字不组合。而这个数字与后一个数字组合的情况，数量就是等于res[i+2], 这个数字与后面一个数字不组合，数量其实就是res[i+1],也就是如果一个数字可以和后一个数字组合时，res[i]=res[i+1]+res[i+2],注意一种特殊情况，就是后面的后面一个数字为0的情况，意味着后面的数字必须与后面的后面一个数字组合，也就是不存在这个数字与后一个数字组合，这是res[i]=res[i+1]。注意处理0的情况，10和20, 当前数字为0时，则res[i]=0，上一个数字为0时，res[i]=res[i+2]。

比较多边界，细节情况，注意调整。

更清晰的表述，refer toDecode Ways – LeetCode
这里可以把范围分成几个区间：
（1）00：res[i]=0（无法解析，没有可行解析方式）；
（2）10, 20：res[i]=res[i-2]（只有第二种情况成立）；
（3）11-19, 21-26：res[i]=res[i-1]+res[i-2]（两种情况都可行）；
（4）01-09, 27-99：res[i]=res[i-1]（只有第一种情况可行）；

Construct Binary Tree from Inorder and Postorder Traversal

Given inorder and postorder traversal of a tree, construct the binary tree.

Note:
You may assume that duplicates do not exist in the tree.

Method

从postorder末尾找到树的的根节点，然后在inorder找到根节点，以此将inorder两分，得到左右子树的size以及推出在postorder，inorder的位置信息，递归建立左右子树赋给根节点。
