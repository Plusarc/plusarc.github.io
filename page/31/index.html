Single Number

Given an array of integers, every element appears twice except for one. Find that single one.

Note:
Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Method

Method 1

使用HashSet，一遍扫描数组，如果元素已经在set中，则从set中移除。
按照题目的条件，扫描完后只会剩下一个只出现一遍的元素，取出返回结果

Method 2

一个很NB的方法(参考)，利用异或位运算的一个性质，即：一个数与自身异或的结果为0。我们只需遍历数组中的每一个元素，并将其进行异或。因为，异或满足交换律，所以最终的异或结果将仅仅包含只出现一次的那个数。

如：1 ^ 2 ^ 2 ^ 1 ^3 ^ 4 ^ 3 = 1 ^ 1 ^ 2 ^ 2 ^ 3 ^ 3 ^ 4 = 4

Maximal Square

Given a 2D binary matrix filled with 0’s and 1’s, find the largest square containing all 1’s and return its area.

For example, given the following matrix:

Return 4.

Method

Naive method

对于每个点，向下向右判断是否满足1 square，知道出现0为止。

Dynamic Programming method

如果一个点(i,j)为0，则其square边长为0.
如果一个点(i,j)为1，则其square边长为min( min(matrix[i][j-1], matrix[i-1][j]), matrix[i-1][j-1])，也就是 (其左边的一个点和上面的一个点的square边长的较小值)，与(左上的一个点square边长)比较的较小值。

原因是以(i,j)为右下角的square，其边长L(i, j) 最多为(i-1,j-1)为右下角的square边长L(i-1,j-1)+1，在满足(i,j-L)到(i,j-1)都为1，以及(i-1,j)到(i-L,j)都为1的情况下。

  1. 若满足(i,j-L)到(i,j-1)都为1，则L(i-1，j)>=L(i-1,j-1)
  2. 若满足(i-1,j)到(i-L,j)都为1，则L(i，j-1)>=L(i-1,j-1)

如果满足上面的两条条件，会有L(i, j)的最大值L(i, j)=L(i-1,j-1)+1.
如果有至少有一条不满足，则L(i, j)的值受限与较短的那条为1的边。

Unique Binary Search Trees

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?

For example,

Given n = 3, there are a total of 5 unique BST’s.

Method

动态规划，每次选择一个值作为根节点，其树的数量为，左子树的数量*右子树的数量。而所有树的数量等于以不同值作为根节点时的树数量的和。

而相同节点数的树的种类都是相同的(结构上)，不论节点的值（在值都是连续的情况下）。

Single Number III

Given an array of numbers nums, in which exactly two elements appear only once and all the other elements appear exactly twice. Find the two elements that appear only once.

For example:

Given nums = [1, 2, 1, 3, 2, 5], return [3, 5].

Note:
The order of the result is not important. So in the above example, [5, 3] is also correct.
Your algorithm should run in linear runtime complexity. Could you implement it using only constant space complexity?

Method

Mehtod 1

使用HashSet，一遍扫描数组，如果元素已经在set中，则从set中移除。
按照题目的条件，扫描完后只会剩下两个只出现一遍的元素，取出返回结果

Mehtod 2

利用位运算，除去出现次数为2次的数(参考)。

如果对所有元素进行异或操作，最后剩余的结果是出现次数为1次的两个数的异或结果，此时无法直接得到这两个数具体的值。但是，因为这两个数一定是不同的，所以最终异或的值至少有一个位为1。我们可以找出异或结果中第一个值为1的位，然后根据该位的值是否为1，将数组中的每一个数，分成两个部分。这样每个部分，就可以采用Single number I中的方法得到只出现一次的数。
