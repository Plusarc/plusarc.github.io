Ugly Number

Write a program to check whether a given number is an ugly number.

Ugly numbers are positive numbers whose prime factors only include 2, 3, 5. For example, 6, 8 are ugly while 14 is not ugly since it includes another prime factor 7.

Note that 1 is typically treated as an ugly number.

Method

如果一个数是ugly number，通过除以一定次数的2, 3, 5，会正好得到结果1(整除得到的，余数为0)。

所以依次在除2余数为0时，将其不断除以2，直到余数不为0。检查结果是否为1，如果是则返回true。如果不是，则在除3余数为0时，将其不断除以3，直到余数不为0。检查结果是否为1，如果是则返回true。如果不是，则在除5余数为0时，将其不断除以5，直到余数不为0。检查结果是否为1，如果是则返回true,如果不是则说明2，3，5无法乘得该数，也就是该数有其他的prime，不是ugly number，返回false。

Unique Binary Search Trees

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?

For example,

Given n = 3, there are a total of 5 unique BST’s.

Method

动态规划，每次选择一个值作为根节点，其树的数量为，左子树的数量*右子树的数量。而所有树的数量等于以不同值作为根节点时的树数量的和。

而相同节点数的树的种类都是相同的(结构上)，不论节点的值（在值都是连续的情况下）。

Single Number II

Given an array of integers, every element appears three times except for one. Find that single one.

Note:
Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Method

Method 1

使用HashMap，一遍扫描数组，如果不在set中，则添加到set中。如果元素已经在set中，且次数为1，则更新为2，如果次数为2，则从set中移除。
按照题目的条件，扫描完后只会剩下一个只出现一遍的元素，取出返回结果

Method 2

使用位操作，尝试消除掉出现3次的数字(参考)。

若一个数字出现3次，则该数字的二进制表示中每个位置的数值为1的出现次数为3次，如：5的二进制表示为101，若5出现3次，则其二进制表示中的第一个位置和第二个位置出现1的次数和均为3。那么对该位1出现的次数对3取余，最终的结果所对应的就是所要求的Single Number。
