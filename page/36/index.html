Convert Sorted List to Binary Search Tree

Given a singly linked list where elements are sorted in ascending order, convert it to a height balanced BST.

Method

递归解决，每次使用快慢指针扫描数组，快指针到链表末尾时，慢指针正好处于链表中间。另外慢指针扫描时，同时将扫描过的链表进行翻转。由于慢指针正好处于链表中间，将其取作根节点，同时用同样的方法递归扫描左子链表构建左子树，扫描右子链表构建右子树。时间复杂度O(nlogn)

Word Break

Given a string s and a dictionary of words dict, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

For example, given
s = “leetcode”,
dict = [“leet”, “code”].

Return true because “leetcode” can be segmented as “leet code”.

Method

Naive method

对于target word进行所有可能的拆分，并检测所有拆分部分是否都在dict中。

DP method

问题可以理解为target word中，targetword.substring(0, length)字符串能否由字典中的词构成。子问题为targetword.substring(0, n)字符串能够由字典中的词构成。

如果targetword.substring(0, n)能够由字典中的词构成，则其必定是由可以由字典中的词构成的子串targetword.substring(0, m)(m<n)，加上字典中的一个词构成的。

加上b(n)表示0-n的子串能够由字典组成，状态转移方程可以写为：

b(n) = (b(m) && dict.contains(targetword.substring(m,n))) || dict.contains(targetword)

Your code should preferably run in O(n) time and use only O(1) memory.

Multiply Strings

Given two numbers represented as strings, return multiplication of the numbers as a string.

Note: The numbers can be arbitrarily large and are non-negative.

Method

首先要注意不能直接转化为数值然后相乘，因为要考虑溢出的问题。

类似与手工乘法，对于结果建立一个数组，每个元素存储结果对应的每一位，如 385 * 97, 个位=5 * 7，十位=8 * 7 + 5 * 9 ，百位=3 * 7 + 8 * 9..., 可以每一位用一个Int表示，存在一个int[]里面。

结果最大长度是num1.len + num2.len，比如99 * 99，最大不会超过10000，所以4位就够了，对应建立相应数组。

先将字符串逆序便于从最低位开始计算，最后结果前面的0要清掉。
