Reorder List

Given a singly linked list L: L0→L1→…→Ln-1→Ln,
reorder it to: L0→Ln→L1→Ln-1→L2→Ln-2→…

You must do this in-place without altering the nodes’ values.

For example,
Given {1,2,3,4}, reorder it to {1,4,2,3}.

Method

首先使用快慢指针遍历list，快指针速度为慢指针的两倍，当快指针到达末尾是，慢指针处于链表中间，停止遍历，慢指针在遍历的同时，将遍历过的子链表翻转，遍历结束得到两个子链表，一个从中间指向开头，一个从中间指向末尾（具体要求位置安需求调整）。

1->2->3->4->5->6

遍历后：
1<-2<-3 and 4->5->6

从两个子链表开头开始，依次移动指针让子链表1的当前指针指向子链表2的当前指针，同时子链表2的当前指针指向上一轮的子链表1的指针(如果是链表长度为偶数，第一轮时指向null，如果链表长度为奇数，同样指向上一轮的子链表1的指针，有些不同，后面说明)。直到两个子链表移动到末尾。注意处理中间信息的保存和更新。

另外链表长度为奇数时，需要多一步处理：

1->2->3->4->5

根据具体的快慢指针设置，这里遍历后得到：

1<-2<-3 and 4->5

两个子链表长度不一致，可以容易知道子链表1的开头节点必然是调整后链表的最后一个节点，设为tail，提前移动子链表1的指针到下一位，后面同样处理，正式处理第一轮子链表2的当前指针不是指向null，而是3,也就是子链表1提前移动过的位置。

For example,
Given nums = [0, 1, 3] return 2.

Note:
Your algorithm should run in linear runtime complexity. Could you implement it using only constant extra space complexity?

Method
