Best Time to Buy and Sell Stock II

Say you have an array for which the ith element is the price of a given stock on day i.

Design an algorithm to find the maximum profit. You may complete as many transactions as you like (ie, buy one and sell one share of the stock multiple times). However, you may not engage in multiple transactions at the same time (ie, you must sell the stock before you buy again).

Method

虽然题目跟Best Time to Buy and Sell Stock类似，但使用的策略是不同的。

首先来看一张股市变化的图：

给定了所有时间的股票价格，最佳策略是什么？ 当然是在局部最低处买入，在同一个对于的下一个局部最高处卖出，其实就是上升子序列，对于所有的上升子序列进行同样的操作。本质上是贪心策略，记入每个上升子序列的非负差值。

所以问题就转化为了检查局部最低处和局部最高处，在局部最低处买入(设置min)，局部最高处卖出(max, profit += (max-min))

Missing Number

Given an array containing n distinct numbers taken from 0, 1, 2, …, n, find the one that is missing from the array.

For example,
Given nums = [0, 1, 3] return 2.

Note:
Your algorithm should run in linear runtime complexity. Could you implement it using only constant extra space complexity?

Method

首先数值序列为0,1,2,3,...n, 其总和为n*(n+1)/2.
对数组进行求和得到sum，则缺的那个数字为 n*(n+1)/2 - sum

House Robber II

Note: This is an extension of House Robber.

After robbing those houses on that street, the thief has found himself a new place for his thievery so that he will not get too much attention. This time, all houses at this place are arranged in a circle. That means the first house is the neighbor of the last one. Meanwhile, the security system for these houses remain the same as for those in the previous street.

Given a list of non-negative integers representing the amount of money of each house, determine the maximum amount of money you can rob tonight without alerting the police.

Method

类似于House Robber问题，区别是最后一个的选择受第一个影响。

也就是当做原来的House Robber问题求解后，结果分为几种情况：

  1. sum计入了num[0]，和num[n]
  2. sum计入num[0]，但没有计入num[n]
  3. sum没有计入num[0]，但计入了num[n]
  4. sum没有计入num[0]，和num[n]

对于后面3种情况，House Robber的解即是House Robber II的解。
唯一不同的是sum同时包含了num[0]和num[n]。这种情况House Robber的解没有办法用于直接作为House Robber的解。

这里我们直接指定分为两类情况计算，一类是计算不计入num[0]的最优解sum，一类是计算计入num[0]的最优解sum，最大sum一定是这两个中的较大值。

不计入num[0]，num[n]的选择不受任何影响，还原为原来的House Robber问题，范围为1...n

计入num[0]时，求得最大sum，又分为sum计入num[n]和没有计入num[n]的情况，对于是否计入了num[n]，直接比较sum[n]与sum[n-1]是否相等即可得知。。没有计入num[n]时，sum可以不做调整，因为没有同时选择第一个和最后一个。计入num[n]时，这样需要去掉nums[0]和nums[n]较小的一个。同时与sum[n-1]比较，取较大值。
