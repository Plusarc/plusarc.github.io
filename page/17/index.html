Candy

There are N children standing in a line. Each child is assigned a rating value.

You are giving candies to these children subjected to the following requirements:

Each child must have at least one candy.
Children with a higher rating get more candies than their neighbors.
What is the minimum candies you must give?

Method

最初实现基于判断波谷，然后向两边递推，时间复杂度为O(N)。效率是比较高的，但是需要处理平行，边界等多种特殊情况，实现十分复杂。

这里参考一种更优更简洁的思路：

初始化所有小孩糖数目为1，从前往后扫描，如果第i个小孩等级比第i-1个高，那么i的糖数目等于i-1的糖数目+1；从后往前扫描，如果第i个的小孩的等级比i+1个小孩高,但是糖的数目却小或者相等，那么i的糖数目等于i+1的糖数目+1。

也就是从前往后扫描后，保证右边ratings更高的孩子一定比左边的孩子糖果数量多。
从后往前扫描后，保证左边ratings更高的孩子一定比右边的孩子糖果数量多。

Max Points on a Line

Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.

Method

每次处理一个点，具体过程：

  1. 对每个点p进行循环扫描（扫描坐标中所有的点），并维护一个duplicate变量，初值为1，表示自己，使用hashmap记录当前点对应斜率的点个数（对于每个点p，需要清空或者新建立一个hashmap）。
  2. 扫描到点q时，当p点与q点重合时，duplicate++。否则计算斜率（注意边界case，斜率为∞），记录斜率为k的直线个数。
  3. 对坐标中所有的点处理完之后，遍历map中的value + duplicate的最大值，有一种特殊情况：所有的点都是重合点，map为空。

Move Zeroes

Given an array nums, write a function to move all 0’s to the end of it while maintaining the relative order of the non-zero elements.

For example, given nums = [0, 1, 0, 3, 12], after calling your function, nums should be [1, 3, 12, 0, 0].

Note:
You must do this in-place without making a copy of the array.
Minimize the total number of operations.

Method

设置一个当前指针，一个实际存储非0元素的指针。将数组的非0元素都移到左边，然后对于将最后非0元素指针及之后的元素全部置0.

Word Search II

Given a 2D board and a list of words from the dictionary, find all words in the board.

Each word must be constructed from letters of sequentially adjacent cell, where “adjacent” cells are those horizontally or vertically neighboring. The same letter cell may not be used more than once in a word.

For example,
Given words = ["oath","pea","eat","rain"] and board =

Return ["eat","oath"].
Note:
You may assume that all inputs are consist of lowercase letters a-z.

Method

最直接的方法就是对所有words中的词都应用Word Search中的算法DFS搜索，将能搜索到的词加入结果集中。

在大量词存在公共前缀情况下，这种方法会造成很多的重复计算，例如某个word不存在，则所以有以这个word为前缀的词也不可能存在于board，没有必要再重复搜索。

先把所有词存入字典树中，然后利用字典树剪枝board的DFS搜索，避免不必要的搜索。

Return a deep copy of the list.

Method

再同时遍历新旧链表一遍，将对于的random指针从根据old node的random指针从HashMap中取出。
